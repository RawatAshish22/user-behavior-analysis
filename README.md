# 📊 User Behavior & Retention Analysis

## 🚀 Project Overview
This project analyzes user behavior, engagement, and spending patterns to identify retention issues and revenue opportunities.

## 🛠 Tools Used
- SQL
- Power BI
- Excel

## 📈 Key Insights
- Majority users (~80%) are inactive → retention issue
- High engagement users contribute most revenue
- Opportunity to convert low-value users

## 📂 Files Included
- dataset.csv → raw dataset
- queries.sql → SQL analysis queries
- dashboard.pbix → Power BI dashboard

## 📊 Dashboard
Built an interactive dashboard to visualize user activity, spending, and engagement patterns.
